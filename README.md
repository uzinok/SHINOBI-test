# Gulp Сборка проекта

Сборка на Gulp для веб-разработки с поддержкой HTML, SCSS, JavaScript, оптимизацией изображений и шрифтов.

## Требования

- Node.js v24.11.1
- npm 11.6.2

## Установка

```bash
npm install
```

или

```bash
npm ci
```

## Доступные команды

### Основные задачи

- `npx gulp start` - запуск сборки для разработки с сервером и отслеживанием изменений
- `npx gulp build` - сборка проекта для продакшена (с минификацией)

### Вспомогательные задачи

- `npx gulp copy` - копирует готовые файлы для билда и при запуске проекта для разработки
- `npx gulp copyMedia` - копирование медиа-файлов во время разработки
- `npx gulp cleanDist` - удаление папки dist
- `npx gulp html` - сборка HTML файлов (поддержка Nunjucks)
- `npx gulp styles` - компиляция SCSS в CSS с автопрефиксами и сортировкой медиа-запросов
- `npx gulp scripts` - транспиляция JavaScript с Babel и минификацией

### Дополнительные задачи

- `npx gulp fonts` - конвертация TTF шрифтов в WOFF и WOFF2 форматы
- `npx gulp optimizeImages` - оптимизация изображений (JPEG, PNG, GIF, SVG)
- `npx gulp sprite` - создание SVG спрайта из иконок в папке icons/sprite
- `npx gulp stack` - создание SVG стека из иконок в папке icons/stack

## Структура папок

```
project/
├── .vscode/*.code-snippets   # Сниппеты для разработки
├── source/                   # Исходные файлы
│   ├── fonts/                # Шрифты (TTF, WOFF, WOFF2)
│   ├── images/               # Изображения
│   │   └── icons/            # SVG иконки
│   │       ├── sprite/       # Иконки для symbol-спрайта (используются через <use>)
│   │       └── stack/        # Иконки для stack-спрайта (используются в CSS через mask/background)
│   ├── scripts/              # JavaScript файлы
│   ├── styles/               # SCSS стили
│   ├── vendor/               # Сторонние библиотеки
│   ├── videos/               # Видео файлы
│   ├── parts/                # HTML/NJK компоненты
│   ├── *.html                # HTML шаблоны
│   ├── *.njk                 # Nunjucks шаблоны
│   └── *.ico                 # Фавиконки
├── dist/                     # Собранный проект (генерируется автоматически)
├── .postcss-sorting.json     # файл настройки для плагина PostCSS Sorting
├── .editorconfig             # настройки проекта (табы, окончания строк...)
└── gulpfile.js               # Конфигурация Gulp
```

Плагин [PostCSS Sorting VSC](https://marketplace.visualstudio.com/items?itemName=mrmlnc.vscode-postcss-sorting). Файл `.postcss-sorting.json` взят из репозитория [htmlacademy](https://github.com/htmlacademy/codeguide/blob/master/.postcss-sorting.json), за что им отдельная благодарность.

## Особенности сборки

- **HTML:** Поддержка Nunjucks шаблонов, минификация HTML
- **CSS:** SCSS компиляция, автопрефиксы, сортировка медиа-запросов, sourcemaps в разработке
- **JavaScript:** Babel транспиляция, terser минификация, строгий режим
- **Изображения:** Оптимизация и создание SVG спрайтов
- **Шрифты:** Конвертация TTF в современные форматы
- **Сервер:** BrowserSync с автоматической перезагрузкой
- **Обработка ошибок:** Визуальные уведомления об ошибках для разных типов задач

## Разработка

Для начала работы выполните:

```bash
npx gulp start
```

Проект будет доступен по адресу `http://localhost:3000`

# Сниппеты для VS Code

## Сниппеты для CSS/SCSS

### `bgimage` - Быстрое добавление фонового изображения

**Префикс:** `bgimage`

**Описание:** Сниппет для быстрой вставки CSS-свойств `background-image` с часто используемыми значениями.

**Генерируемый код:**

```scss
background-image: url(${1});
background-position: ${2:center};
background-repeat: ${3:no-repeat};
background-size: ${4:contain};
```

**Использование:**

1. Введите `bgimage` и нажмите *Tab*
2. Укажите путь к изображению
3. Настройте позиционирование (по умолчанию: `center`)
4. Настройте повторение (по умолчанию: `no-repeat`)
5. Настройте размер (по умолчанию: `contain`)

*Переход между настройками осуществляется с помощью `TAB`*

### `fontface` - Подключение локального шрифта

**Префикс:** `fontface`

**Описание:** Сниппет для подключения локальных шрифтов через `@font-face` с автоматическим использованием содержимого буфера обмена.

**Генерируемый код:**

```scss
@font-face {
    font-family: '${1:$CLIPBOARD}';
    font-weight: ${2:400};
    font-style: ${3:normal};
    font-display: ${4:swap};
    src: url(${5:../../fonts}/${6:$CLIPBOARD}.woff2) format('woff2'),
         url(${5:../../fonts}/${6:$CLIPBOARD}.woff) format('woff'),
         url(${5:../../fonts}/${6:$CLIPBOARD}.woff) format('truetype');
}
```

**Использование:**

1. Введите `fontface` и нажмите *Tab*
2. Попавьте название шрифта
3. Настройте жирность шрифта (по умолчанию: `400`)
4. Настройте наклон (по умолчанию: `normal`)
5. Настройте оптимизацию загрузки шрифта (по умолчанию: `swap`)
6. При необходимости поправьте пути

**Особенности:**

- Автоматически использует название шрифта из буфера обмена
- Поддерживает современные форматы шрифтов (*WOFF2*, *WOFF*)
- Включает настройки `font-display` для оптимизации загрузки
- Предлагает гибкие настройки веса и стиля шрифта

**Примечание:** Перед использованием скопируйте *название шрифта без расширения* в буфер обмена для автоматической подстановки.
